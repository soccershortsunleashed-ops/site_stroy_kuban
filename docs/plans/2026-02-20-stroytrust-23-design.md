# StroyTrust-23 Website Design

## 1. Контекст

Сайт «СтройТрест-23» уже реализован как многостраничный корпоративный продукт на Next.js + Shadcn UI.

Цель текущей итерации: усилить визуальную подачу через бизнес-анимацию без потери читаемости, производительности и делового характера.

## 2. Утвержденное направление

- Режим: **иммерсивный**
- Область внедрения: **Главная + Процесс + Проекты**
- Приоритет: **баланс визуала и производительности**
- Языковая политика: **все пользовательские тексты только на русском**

## 3. Принципы анимации

1. Ритм движения:
- Фоновые движения медленнее контента
- Интерфейсные реакции быстрее фоновых эффектов

2. Глубина:
- На ключевых экранах 3 слоя: фон, контент, акцент
- Разная скорость смещения слоев при прокрутке

3. Локальное закрепление:
- Закрепляем только опорные секции
- Избегаем глобальной «тяжелой» скролл-сцены на всей странице

4. Длительности:
- Появления блоков: 0.45-0.7 с
- Переходы состояний: 0.2-0.35 с
- Без резких ускорений и дерганых кривых

## 4. Раскладка по страницам

### 4.1 Главная

- Закрепленный первый экран на коротком участке прокрутки
- Ступенчатое появление заголовка, подзаголовка и действий
- Анимация счетчиков метрик при входе в viewport
- Каскадное появление карточек направлений
- Плавное раскрытие графиков (однократный запуск при первом показе)

### 4.2 Процесс

- Вертикальная линия прогресса, растущая по прокрутке
- Активный этап визуально выделен, соседние приглушены
- Изображение этапа раскрывается плавно, без резких переходов
- Переходы между этапами через мягкое перекрытие

### 4.3 Проекты

- Фокус на активном проекте при прокрутке списка
- Активная карточка слегка увеличена, соседние менее контрастны
- Фото внутри карточек с очень медленным «живым кадром»
- Детали проекта появляются поэтапно с небольшой задержкой

## 5. Мобильные ограничения и доступность

1. На мобильных:
- Убираем длинные закрепления
- Снижаем глубину параллакса
- Сохраняем короткие анимации появления и подсветку активного шага

2. Для prefers-reduced-motion:
- Отключаем скролл-управляемые эффекты
- Оставляем мгновенные или минимальные переходы

## 6. Ограничения реализации

- Все UI-компоненты берутся из Shadcn-экосистемы
- Анимации реализуются поверх существующих секций, без переписывания структуры страниц
- Контент остается строго русскоязычным
- Без деградации сборки (`npm run lint`, `npm run build` обязательны)

## 7. Критерии приемки

- Визуально заметная, но деловая анимационная глубина на 3 страницах
- Плавный скролл без критических просадок на типовых мобильных устройствах
- Корректная работа в режиме reduced motion
- Отсутствие англицизмов в пользовательских текстах
